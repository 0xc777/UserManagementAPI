# UserManagementAPI
## Описание
Демонстрация навыков цикла разработки и автоматизации тестирования, включая проектирование, реализацию API, работу с базой данных и генерацию отчётов.
## Основные функции
- Создание пользователя через REST API
- Проверка данных в базе данных после создания пользователя
- Удаление пользователя
- Интеграционные тесты и генерация наглядных отчётов с помощью Allure
  
- ## Используемые технологии
- Java
- Maven 
- JUnit5
- Allure
- SQLite 
- GitHub
  
## Что реализовано
- проектирование структуры данных и API
- реализация логики создания пользователя, поиск пользователя, удаление
- написание автоматизированных тестов
- интеграцию с инструментами отчётности (Allure)
